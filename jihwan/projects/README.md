## Toby-Spring
토비의 스프링 예제를 위해 SpringBoot 프로젝트를 생성했습니다.

## 환경 셋팅 및 필수 조건
SpringBoot 환경설정에는 다음과 같습니다.

- Language: Java 11
- SpringBoot:2.3.x
- Packaging: Jar

> 주의: SpringBoot: 3.0.x 이상이면 Java 17 이상 사용해야 합니다.

## SpringBoot 생성하기
1. https://start.spring.io 으로 접속합니다. 

2. **프로젝트 선택**을 다음과 같습니다. 
   1. Projects: Gradle-Groovy
   2. Spring: 2.3.x 이상
   3. Language: Java
   4. Packaging: Jar
   5. Java: 11
3. 프로젝트를 생성합니다.
